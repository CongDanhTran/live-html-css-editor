TCP stands for Transmission Control Protocol, creates connections between clients/severs with network interfaces, is the bone of the Internet. Messages transfered by TCP procotol are splitted into smaller packets then reassemble on the other side

HTTP stands for HyperText Transfer Protocol,is an application layer running over TCP protocol, allows communications between clients and servers
